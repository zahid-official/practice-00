Border

The CSS border properties allow you to specify the style, width, and color of an element's border. The border-style property specifies what kind of border to display.

The border property is a shorthand property for the following individual border properties: 1.border-width
2.border-style (required)
3.border-color

The border-radius property is used to add rounded borders to an element

The border-style property can have from one to four values (for the top border, right border, bottom border, and the left border).

The border-width property specifies the width of the four borders. It can be set as a specific size (in px, pt, cm, em, etc) or by using one of the three pre-defined values: thin, medium, or thick

The border-color property is used to set the color of the four borders.If border-color is not set, it inherits the color of the element
