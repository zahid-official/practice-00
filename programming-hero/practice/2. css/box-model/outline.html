Outline

An outline is a line that is drawn around elements, OUTSIDE the borders, to make the element "stand out". It has four properties: 1.outline-style 2.outline-color 3.outline-width 4.outline-offset

Note: Outline differs from borders! Unlike border, the outline is drawn outside the element's border, and may overlap other content. Also, the outline is NOT a part of the element's dimensions; the element's total width and height is not affected by the width of the outline.

The outline-offset property adds space between an outline and the edge/border of an element. The space between an element and its outline is transparent.

Mixed Outline Not supported

Note: None of the other outline properties will have ANY effect unless the outline-style property is set!
