This is Background Blend

The background-blend-mode property defines the blending mode of each background layer (color and/or image)

By default the blending mode sets to normal. In this mode the background-color will not bleed through the background-image.

This is Background Multiply

In this mode the background-color are multiplied and typically this leads to a darker image than before.

This is Background screen

In this mode both image and color is inverted, multiplied and then inverted again

This is Background overlay

In this mode the background-color is mixed with the background-image to reflect the lightness or darkness of the backdrop

This is Background darken

In this mode if the background-image is darker than the background-color then the image is replaced, otherwise it is left as it was

This is Background lighten

In this mode if the background-image is lighter than the background-color then the image is replaced, otherwise it is left as it was.

This is Background color-dodge

In this mode the background-color is divided by the inverse of the background-image. This is very similar to the screen blend mode.

This is Background saturation

In this modekeeps the saturation of the background-image whilst mixing the hue and luminosity of the background-color.

This is Background color

In this mode keeps hue and saturation of background-image and the luminosity of the background-color. In this example because the image is gray and the effect preserves gray levels all we get is a big gray blob

This is Background luminosity

In this mode luminosity of the top color is preserved whilst using the saturation and hue of the background-color.

This is Background luminosity

In this mode luminosity of the top color is preserved whilst using the saturation and hue of the background-color.
