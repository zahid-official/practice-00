This is Background Size

The background-size property specifies the size of the background images.

There are four different syntaxes you can use with this property: the keyword syntax ("auto", "cover" and "contain"), the one-value syntax sets the width of the image (height becomes "auto"), the two-value syntax (first value: width of the image, second value: height), and the multiple background syntax (separated with comma)

by Default The, background image size is auto.It displayed background image in its original size

This is Background Size Cover

Resize the background image to cover the entire container, even if it has to stretch the image or cut a little bit off one of the edges

This is Background Size contain

Resize the background image to make sure the image is fully visible

This is Background Size Set Value of Height and Width

Sets the width and height of the background image in percent or pixel. The first value sets the width, the second value sets the height. If only one value is given, the second is set to "auto"
