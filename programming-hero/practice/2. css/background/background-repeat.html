Background Image Repeat

By default, the background-image property repeats an image both horizontally and vertically.

Background Image Repeat Horizontally

If the image repeated only horizontally set background-repeat: repeat-x;.

Background Image Repeat Vertically

To repeat an image vertically, set background-repeat: repeat-y;

Background Image No Repeat

Showing the background image only once is also specified by the background-repeat: no-repeat

Background Image Repeat Space

The background-image is repeated as much as possible without clipping. The first and last image is pinned to either side of the element, and whitespace is distributed evenly between the images

Background Image Repeat Round

The background-image is repeated and squished or stretched to fill the space (no gaps)
