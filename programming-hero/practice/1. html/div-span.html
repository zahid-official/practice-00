A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.A block-level element always takes up the full width available (stretches out to the left and right as far as it can).The div element defines a division or a section in an HTML document.

  • A block-level element always starts on a new line and takes up the full width available
  • An inline element does not start on a new line and it only takes up as much width as necessary
  • The div element is a block-level and is often used as a container for other HTML elements
  • The span element is an inline container used to mark up a part of a text, or a part of a document

There are different methods for aligning elements side by side

  • Float
  • Display: flex
  • Inline-block
  • Grid
