The primary purpose of an anchor tag is to link one page to another page or to a section of the same page. The anchor tag is also known as a HyperLink. For use this anchor tag to link to a web page. You need to use the href attribute to link to another page.The value of the href attribute is usually a URL pointing to a web page.
